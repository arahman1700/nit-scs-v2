   - Add `RECONCILIATION_THRESHOLD` to the Zod schema: `RECONCILIATION_THRESHOLD: z.coerce.number().min(0).default(0.001)` -- this is the minimum absolute quantity difference to flag as a discrepancy (default 0.001 matches Decimal(12,3) precision)
   - This makes the threshold configurable via environment variable without code changes

2. **Update `packages/backend/src/domains/scheduler/jobs/maintenance-jobs.ts`**:
   - Import `getEnv` from `../../../config/env.js`
   - In `runDailyReconciliation(ctx)`:

   a. **Add threshold filtering**: After computing `discrepancies` (the array with diff values), filter by threshold:
      ```typescript
      const threshold = getEnv().RECONCILIATION_THRESHOLD;
      const significantDiscrepancies = discrepancies.filter(d => Math.abs(d.diff) > threshold);
      ```
      Replace all downstream uses of `discrepancies` with `significantDiscrepancies`.

   b. **Remove auto-correction**: The current code auto-corrects inventory levels (lines ~330-335 with `ctx.prisma.inventoryLevel.update`). Remove the auto-correction loop. Instead, only LOG and NOTIFY about discrepancies. The reconciliation job should DETECT and FLAG, not silently fix. Change the log message from "auto-corrected" to "flagged":
      ```typescript
      ctx.log('warn', `[Reconciliation] ${significantDiscrepancies.length} discrepancies above threshold ${threshold}`, {
        discrepancies: significantDiscrepancies.slice(0, 10),
      });
      ```

   c. **Add audit trail**: After notification, create audit log entries for each discrepancy:
      ```typescript
      for (const d of significantDiscrepancies) {
        await ctx.prisma.auditLog.create({
          data: {
            tableName: 'InventoryLevel',
            recordId: d.itemId,
            action: 'reconciliation_discrepancy',
            performedAt: new Date(),
            changes: {
              warehouseId: d.warehouseId,
              expected: d.lotTotal,
              actual: d.qtyOnHand,
              diff: d.diff,
              threshold,
            },
          },
        });
      }
      ```

   d. **Log summary even when no discrepancies**: After the check, always log the result:
      ```typescript
      ctx.log('info', `[Reconciliation] Completed: ${significantDiscrepancies.length} discrepancies above threshold ${threshold} (${discrepancies.length} total, ${discrepancies.length - significantDiscrepancies.length} below threshold)`);
      ```
