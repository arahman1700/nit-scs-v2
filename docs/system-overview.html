📦 Module 1 — Warehousing & Inventory Management

M1-F01Material Receiving (GRN)
Phase 1

Digital Goods Receipt Note for recording inbound materials against Purchase Orders. Auto-generates OSD reports for variances, triggers QC inspection with 4-hour SLA, auto-updates bin card balances on approval.

Key fields: GRN Number (auto), Warehouse, PO Number, Supplier, Invoice, Delivery Note, Items (Code, Desc, UOM, PO Qty, Received Good, Received Damaged, Over/Short), Bin Location, QC Result, Timestamps

Format: GRN-YYYY-NNNNN

M1-F02Material Issue to Projects (MIRV)
Phase 1

Digital Material Issue Request Voucher. Real-time stock availability check blocks over-issue. Multi-level approval: Technical Manager → WH Manager (≤SAR 200K) → SC Manager (>SAR 200K). Auto-generates pick list and gate pass. Delivery confirmation with digital signature decrements inventory and allocates cost to project.

Format: MIRV-YYYY-NNNNN | Threshold: SAR 200,000

M1-F03Stock Transfer Between Warehouses (IMSF)
Phase 1

Internal Material Shifting Form. Dual approval from source and destination WH Managers. QC inspection of transferred materials. Auto-linked gate pass. Source decrements on approval, destination increments on receipt confirmation. OSD auto-generated for quantity mismatches.

M1-F04Cycle Count & Physical Inventory
Phase 2

Scheduled counting (monthly cycle + annual full). Blind count sheets hide system quantities. 5% variance threshold triggers NCR. Stock Adjustment requires WH Manager approval. Auto-generated accuracy reports.

M1-F05Bin Card Management
Phase 1

Computed real-time views (NOT manually maintained). Running balance = sum of all transactions per item per bin. Auto-updated by GRN, MIRV, IMSF, MRV, and adjustments. Searchable by item, warehouse, bin location. Date range filtering.

M1-F06Material Returns from Projects (MRV)
Phase 2

Linked to original MIRV. QC condition assessment: Good → re-stock, Damaged → quarantine, Unusable → scrap (M1-F08). Photo documentation required for non-Good items. Full traceability: issued → returned → re-shelved or disposed.

M1-F07Tools Issue & Return
Phase 3

Digital checkout linked to employee. Auto-alerts for overdue returns. Condition tracking (issued vs. returned). Maintenance scheduling based on usage.

M1-F08Scrap & Waste Disposal
Phase 1

Flag items by reason (expired, damaged, obsolete). Mandatory photos. QC assessment → Decision (Scrap/Sell/Return to Supplier/Reclassify). WH Manager approval. Auto-decrement on disposal. Quarterly scrap summary report.

M1-F09Minimum Stock Alerts & Reordering
Phase 3

Configurable minimum stock per item per warehouse. Daily auto-check. Alerts to Inventory Specialist + WH Manager. Consumption trend report (avg daily consumption from historical issues).

M1-F10Warehouse Handover & Shift Management
Phase 3

Digital handover form mandatory at shift change. Auto-populates pending issues, in-progress transactions, active alerts. Both outgoing + incoming sign off digitally. Searchable history.

M1-F11Warehouse Compliance & Safety Audits
Phase 3

Digital audit checklists (TUV, SGS, ISO 9001). Findings: Conforming/Minor NC/Major NC/Observation. Corrective action plans with owners and deadlines. Follow-up reminders.

M1-F12Annual Maintenance Contracts (AMC)
Phase 3

Contract storage (vendor, dates, scope, cost). Auto-renewal alerts 30 days before expiry. Maintenance scheduling linked to contracts. Cost tracking per warehouse.

M1-F13Inventory Dashboard
Phase 1

Real-time stock levels. Material movement summary (GRN/MIRV/IMSF/MRV). Accuracy metrics from cycle counts. Drill-down to individual transactions. Filterable by warehouse, date, category.

M1-F14QC Inspection Workflow
Phase 1

Cross-cutting QC for GRN, MIRV, MRV, IMSF. Results: Approved/Rejected/Quarantine. Photo attachments (max 5, 5MB each). 4-hour SLA notification to QC Inspector. Result updates parent transaction.

🔧 Module 2 — Equipment & Transportation Management

M2-F01Equipment Rental Request (Job Order)
Phase 2

Request form for Heavy Equipment/Generator/AC. Auto-pulls rate card. Cost calc: (rate × duration) + transport + operator + fuel + insurance. Approval: WH Mgr ≤200K, SC Mgr >200K. Return date alerts.

M2-F02Equipment Delivery & Return
Phase 2

Delivery note with condition photos + hours/mileage. PM digital sign-off starts rental period. Auto-cost accrual. Return note with side-by-side condition comparison. Damage cost tracking. Final cost = actual days + damages + fuel.

M2-F03Transportation of Materials
Phase 2

Transport request: origin, destination, load, vehicle type. Vehicle/driver assignment. Auto-linked gate pass. Status: Loading → In Transit → Delivered. Cost per trip linked to project.

M2-F04Supplier Performance Evaluation
Phase 3

Auto-collected metrics: on-time delivery, equipment quality, pricing compliance, responsiveness, safety. Weighted scoring. Decision: Continue/Improve/Terminate. Historical trends.

M2-F05Rental Agreements & Rate Cards
Phase 2

Rate card records with daily/weekly/monthly rates. Auto-referenced by job orders. Expired cards flagged. Deviation requires approval. Rate history. Quarterly review scheduling.

M2-F06Emergency Equipment Requests
Phase 3

Emergency request with mandatory justification. Routes directly to SC Manager (bypasses WH Manager). Premium cost tracking. Post-incident review.

M2-F07Equipment Dashboard
Phase 4

Active rentals with status, location, cost-to-date. Costs by project. Supplier performance scores. Drill-down to job orders.

🚢 Module 3 — Shipping & Customs Clearance

M3-F01Shipment Notification & Tracking
Phase 2

7-stage lifecycle: Booked → In Transit → At Port → In Customs → Released → In Transport → Delivered. Carrier types: Sea/Air/Land. Overdue auto-flagging.

M3-F02Customs Documentation
Phase 3

Auto-generated document checklist. HS Code classification per item (reusable). Manages: Commercial Invoice, Packing List, COO, BOL, Insurance.

M3-F03Import Clearance Process
Phase 3

Step-by-step clearance tracking. Auto-notifications on customs requests. Physical inspection tracking. Hold/reject with reason documentation.

M3-F04VAT & Duties Management
Phase 3

Duties auto-calculated from HS codes. VAT at KSA 15%. Finance User approval for payments. Cost auto-allocated to projects.

M3-F05Shipment Release Authorization
Phase 3

Verifies: docs approved + inspection passed + duties paid + VAT paid. SC Manager authorizes release. Triggers transport order to warehouse → GRN.

M3-F06Shipment Dashboard
Phase 4

All shipments by status. Overdue highlighting. Drill-down to individual records. Filterable by project, carrier, date.

⚙️ Module 4 — Support & Administration

M4-F01Master Data Management
Phase 1

CRUD for Warehouses (75), Items, Suppliers, Projects. Sys Admin only. Soft-delete to preserve referential integrity. All changes logged.

M4-F02Employee Management
Phase 1

Employee records: name, ID, position, department, warehouse assignment, qualifications, training. Role-based access. Row Owner assignment. Onboarding/offboarding workflows.

M4-F03User & Role Management
Phase 1

14 role-based access levels. Row Owner scoping per warehouse. User ↔ Employee link. Password management, session management, audit logging.

M4-F04Approval Engine
Phase 1

Multi-step workflows for MIRV, JO, Stock Adj, Scrap, Shipment Release, Customs Payment, Emergency Equipment. 4-hour SLA default. Configurable: SLA durations, recipients, thresholds. NOT configurable: Adding/removing steps (requires Change Request).

M4-F05Notification Engine
Phase 1

13 notification types. Push + In-app. Delivery within 5 seconds. Read/unread status. Users can mute non-critical; critical notifications cannot be muted.

M4-F06Reporting & Analytics
Phase 1

Dashboards for Inventory, MIRV/GRN, Gate, Equipment, Shipments. Drill-down paths defined per SOW. Row Owner restrictions apply.

M4-F07Scheduled Reports
Phase 4

Daily, weekly, monthly, quarterly reports. In-app + CSV export.

M4-F08Compliance & Audits
Phase 4

Digital audit checklists. NCR with corrective action plans. Auto-reminders. Follow-up verification. Compliance dashboard.

M4-F09Maintenance Tracking
Phase 4

Work orders, technician assignment, execution recording. Preventive maintenance with auto-alerts.

M4-F10Full KPI Dashboard
Phase 4

15 KPIs with current vs. target. Trend display (monthly). Quarterly business review. CSV export.

M4-F11Administrative Support
Phase 3

System config management. Audit log viewer. Bulk CSV export. System health page.

🚧 Module 5 — Gate Pass & Access Control

M5-F01Outbound Gate Pass
Phase 1

Gate Officer looks up MIRV/IMSF/JO by reference. Verifies physical load matches digital authorization. No valid auth = deny + notify WH Manager + log. Captures driver, vehicle, materials, timestamp.

M5-F02Inbound Gate Pass
Phase 1

Check delivery note + PO. Verify against expected deliveries (pending GRNs/shipments). Log entry, direct to warehouse, link to subsequent GRN.

M5-F03Visitor Management
Phase 2

Digital registration, host approval, visitor pass. Timed visits with 8-hour overstay alerts. Exit logging. Searchable history.

M5-F04Material Movement Tracking
Phase 1

Every gate movement linked to source transaction. Quantity verification. Daily auto-reconciliation (gate vs inventory). Mismatches flagged immediately.

M5-F05Access Control & Security
Phase 3

Authorized personnel list. Unauthorized persons flagged. Security Incident Reports with evidence. Investigation tracking. Access pattern analysis.
